## 📂 **Instalación y Uso**
### 1️⃣ **Clonar el repositorio**
```sh
git clone https://github.com/ChrisLCoop/portfolia_v1_react.git
cd christian_limas_portafolio
```

### 2️⃣ **Instalar dependencias**
```sh
npm install
```

### 3️⃣ **Ejecutar en desarrollo**
```sh
npm run dev
```

### 4️⃣ **Construir para producción**
```sh
npm run build
```
